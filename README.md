Follow the steps below to set up and run the project locally:

1. **Clone the repository**
	git clone https://github.com/RushikeshKulkarni07/Hospital-Management-System/blob/main/README.md

	python -m venv venv

		venv\Scripts\activate

	python app.py

	http://127.0.0.1:5000
